# Source Field

This document defines the role of the source-field scaffold in this repo.

## Purpose

The `source_field.py` module provides a minimal queue-and-manifest model for deferred question handling.

It is intentionally small:

- questions can be submitted
- questions receive a stable short hash
- a placeholder manifestation can be produced later

This is not a scientific claim. It is the package compartment for the source-field concept that previously lived only in README prose.
